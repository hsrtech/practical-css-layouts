Practical Layouts with CSS Grid

Best way to learn something new is to practice. While i started learning CSS Grid Module i thought i should create some real pages to get real challenges with Grid Module

This is an open source project so that anybody can utilise the code for learning.

FAQ

  • Can i use this code? This is an open source project, you can use the code as you like.
  • Can i contribute? Yes, any contributions are welcome, if they are relevant to this project
  • Can i provide you a design for coding? Yes, but all pages/designs will be available as part of this open source project. You can hire me for your personal projects.
